> # bookinfo_manage
使用Java+JDBC+JSP+Servlet实现的图书管理系统

## 关于版本
* Java-1.8.0_281
* Tomcat 9.0
* mysql-connector-java-8.0.23(MYSQL 8.0.23)
* jquery-3.5.1
* bootstrap-3.3.7
* jakarta-taglibs-standard-1.1.2(jstl.jar、standard.jar)

## 功能列表
* 显示图书列表
* 添加图书信息
* 修改图书信息
* 删除图书信息

## 反思
* Servlet使得代码量冗长
* 实现图书列表的分页功能较麻烦
* 没有实现数据库报错在页面的显示
